# bookbot

BookBot is my first [Boot.dev](https://www.boot.dev) project!

## Usage

This program can take the argument as a relative path to a text file and will display the number of words in that text file and the counts for each alphabetical character in that text file. 

1. have python 3 installed
2. copy this repo locally
3. navigate to the bookbot directory using your terminal/command line
4. use the following to execute the program
     `Usage: python3 main.py <path_to_book>`
